# 🚀 Guia Rápido de Uso - Sistema Otimizado

## ⚡ Operação Básica com Teclado

### 1. Abrir Mesa
```
1. Clicar na mesa desejada (ou navegação)
2. Modal "Pedido" abre automaticamente
```

### 2. Adicionar Itens RÁPIDO (Nova Abordagem)
```
1. Clicar na aba "Rápido" (⚡ ícone)
2. Campo "Código" já focado:
   
   Digitar: 100 (Coca-Cola)
   Pressionar: ENTER
   
   ↓
   
   Campo "Quantidade" focado:
   Digitar: 2
   Pressionar: ENTER
   
   ↓
   
   Campo "Observações" focado:
   (opcional, deixar em branco é OK)
   Pressionar: CTRL+ENTER para confirmar
   
   ✅ Item adicionado!
```

### 3. Fechar Conta
```
1. Aba "Fechar" no modal
2. Escolher forma de pagamento:
   - Dinheiro (verde)
   - PIX (azul)
   - Cartão (roxo)
3. Pressionar botão "Fechar Conta"
4. Mesa liberada para novo cliente
```

## ⌨️ Atalhos de Teclado

### No Modal de Pedido:
```
TAB             → Alternar entre campos e abas
ENTER           → Confirmar na sequência (Código → Qtd → Observação)
ESC             → Fechar modal
CTRL+ENTER      → Confirmar adição com observação
Backspace (vazio) → Voltar ao estágio anterior
```

### Na Tela de Mesas:
```
CLIQUE          → Abrir mesa
DUPLO CLIQUE    → Fechar mesa rápido
ESC             → Sair do modal
```

```

13. Selecionar forma de pagamento
```

## 💡 Dicas para Operação Rápida

1. **Memorizar códigos principais** (2, 3, 4, 100, 101)
   - Ganha 5 segundos por item

2. **Não usar mouse** quando possível
   - Use TAB para navegar
   - Use ENTER para confirmar
   - Use ESC para cancelar

3. **Observações rápidas**
   - "S/cebola" (sem cebola)
   - "Ao ponto" (carnes)
   - "Extra" (bebidas maiores)

4. **Verificar total antes de fechar**
   - Aba "Pedido" mostra valores em tempo real
   - Subtotais por item

5. **Manter mesas abertas enquanto cliente está comendo**
   - Não é necessário fechar para adicionar mais

## ❌ Erros Comuns & Soluções

### "Produto não encontrado"
```
❌ Problema: Digitou código errado
✅ Solução: 
   - Conferir tabela de códigos
   - Ou ir para aba "Cardápio" para buscar
   - Ou criar novo produto em Produtos
```

### "Não consigo sair da observação"
```
❌ Problema: Pressionou ENTER
✅ Solução: 
   - Use CTRL+ENTER para confirmar
   - Ou ESC para cancelar
```

### "Mesa não abre quando clico"
```
❌ Problema: Talvez internet lenta ou cache
✅ Solução: 
   - Pressione F5 para recarregar
   - Feche e abra navegador
```
